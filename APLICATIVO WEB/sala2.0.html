Esta página muestra un itineario de todo lo que posee la sala de informática, lo que sirve y lo que ya no funciona, la cantidad que posee cada objeto de las dos aulas y el número de serie que tienen, luego se implementará cuando se registró la última vez que se usó el aula, toda esta información se obtuvo de secretaria académica, otorgada por la almacenista Rosita Milia Serrano Artunduaga.
